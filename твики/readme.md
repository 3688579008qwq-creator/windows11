# Твики
Для выполнения твика запустите файл "**Start Script**". 
<br>
В папке "**PowerRun**" хранится исходный код скрипта, и оболчка для получения прав NT-SYSTEM
<br><br>
Весь исходный код доступен для редактирования и создания своих твиков, безопаснее чем если бы Вы использовали сторонее ПО
<br>
